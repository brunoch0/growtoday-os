# Growtoday OS · Solopreneur Multi-Venture Dashboard

> Founder OS for running 18 business units as one person.
> Personal track feeds Business track. Tools graduate from learning to production.

**Built by**: Bruno Cho ([@DubaiBruno](https://youtube.com/@DubaiBruno))
**Operating**: [Growtoday Holdings FZE](https://growtodayholdings.com) · Dubai
**Build date**: May 2026

---

## What this is

A single-file HTML dashboard that visualizes a real solopreneur's entire operating system:

- **Personal layer** (Operator state) — workout, study, tool lab, habits
- **Business layer** (System) — 18 ventures across Founder IP, Character IP, Marketing, Media, Consulting, Web3, Pet, Defense
- **Pipeline visualization** — how personal learning graduates into business execution
- **L1 drill-down** — full funnel breakdown (MoraèVision implemented as proof-of-concept)
- **Live simulation** — animated Zellys characters walking through workflows in real time

Open `index.html` in any browser. No server, no build, no deps.

- **Defense AI** — custom military-saluting Zellys GIF

## Tech Stack

Single HTML file. No framework. No build step.

- Vanilla HTML + CSS + JS
- Inline SVG brand logos (from [Simple Icons](https://simpleicons.org/))
- Heroicons solid (functional icons)
- Base64-embedded PNG/GIF sprites (Zellys characters)
- IBM Plex Sans + IBM Plex Mono (Google Fonts)

Total size: ~1.5 MB. Self-contained — works offline.
